Workshop cl

oud computing dengan menggunakan Amazon Web Service (AWS) Cloud.
Pada saat workshop, peserta akan dibawa untuk menjawab pertanyaan-pertanyaan seperti apa itu cloud computing, kenapa cloud computing, dan bagaimana menggunakan cloud computing khususnya AWS.

Peserta akan mendapat hands on beberapa layanan yang disediakan oleh AWS seperti, Elastic Compute Cloud (EC2) yaitu layanan komputasi di cloud, Elastic Block Storage (EBS) yaitu layanan penyimpanan data di dalam disk, Relational Database Service (RDS) MySQL yaitu layanan database relasional contohnya MySQL, and Elastic Load Balancer (ELB) yaitu layanan untuk membagi traffic.

Setelah memperoleh pengetahuan dasar layanan-layanan yang disediakan oleh AWS, peserta akan dibagi ke dalam kelompok-kelompok dan diminta untuk menggabungkan layanan-layanan yang telah dikenalkan sebelumnya menjadi satu kesatuan. Sebagai contoh EC2, EBS, RDS, dan ELB dapat digabungkan untuk menjalankan aplikasi web berbasis PHP dimana semua komponen tahan terhadap kegagalan (No Single Point of Failure).

Hasil akhir dari workshop ini diharapkan peserta mengetahui manfaat cloud computing dan dapat menggunakan cloud computing untuk menyelesaikan berbagai masalah yang berkaitan dengan komputasi dengan lebih mudah dan cepat.

Pembicara:

  • Roy Inganta Ginting | Senior System Engineer | GDP Labs
  • Petry Sianipar | Product Manager | GDP Labs

Pelaksanaan:

Hari Jumat, 10 April 2015 Pukul 08.00 – 17.00

Tempat: GD513

Peserta: Mahasiswa Prodi TK Tingkat III (Wajib), Mahasiswa Prodi TK Tingkat II (Optional)
